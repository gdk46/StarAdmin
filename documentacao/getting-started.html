Inicie

Star CSS é uma framework CSS leve, responsivo e moderno para um desenvolvimento mais rápido e extensível.

Star CSS fornece estilos básicos para tipografia e elementos, sistema de layout responsivo baseado em flexbox, componentes e utilitários CSS puros com codificação de melhores práticas e linguagem de design consistente.
