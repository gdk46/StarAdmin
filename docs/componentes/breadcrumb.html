Breadcrumb

Indica a localização da página atual dentro de uma hierarquia de navegação que adiciona separadores automaticamente via CSS.

Caso de uso

O componente Breadcrumb consiste em links, que são alinhados lado a lado e separados por um divisor. Dessa forma é usado para criar uma hierarquia entre as páginas e mostrar o conteúdo.
